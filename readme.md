🎯 AI Mock Interview Platform

An AI-powered interview preparation platform where users upload their resume, receive role-specific interview questions generated by GPT-4o-mini, answer them within a time limit, and get detailed feedback scored on confidence, communication, and correctness.

🔗 Live API: https://backendaiagent.onrender.com
📬 Postman Docs: https://documenter.getpostman.com/view/52202593/2sBY4HT3n9
🚀 Features


Resume Parsing — Upload a PDF resume; AI extracts your role, experience, skills, and projects automatically
AI Question Generation — GPT-4o-mini generates 5 personalized interview questions based on your resume and selected mode (HR or Technical)
Timed Answer Submission — Each question has a time limit; answers submitted after the limit are auto-rejected
Per-Question AI Scoring — Every answer is scored on confidence, communication, and correctness (0–10 each)
Interview Report — Full breakdown of scores per question with human-style feedback after the interview ends
Credit System — Users start with 100 free credits; each interview costs 50 credits
Razorpay Payments — Buy more credits via Razorpay with full payment verification and fraud prevention
Google Auth — One-click sign in with Google; JWT stored in secure HTTP-only cookies
Interview History — View all past interviews with final scores and status

🔐 Security Implementation


JWT in HTTP-only cookies — tokens are never accessible via JavaScript, preventing XSS attacks
Ownership checks — every interview route verifies the requesting user owns that interview before proceeding
Rate limiting — AI routes are limited to 10 requests/minute; auth routes to 20 requests/15 minutes to prevent brute force and credit abuse
Razorpay signature verification — payments are verified using HMAC SHA-256 before credits are added
Idempotency — duplicate payment webhooks are detected and rejected
Environment-aware error messages — stack traces are hidden in production, shown in development

💡 Key Implementation Details

Safe AI Response Parsing
AI responses sometimes return JSON wrapped in markdown code fences. A custom safeParseAI utility strips these fences before parsing, preventing crashes from malformed responses.

Score Calculation
Each answer is scored on three dimensions by the AI. The calcAverages utility computes the overall interview score across all questions — reused in both the finish and report endpoints to keep logic consistent.

PDF Text Extraction
Resumes are parsed using pdfjs-dist to extract raw text page by page. The text is cleaned and injected into the AI prompt to generate questions tailored to the candidate's actual experience.

Credit Deduction Safety
Credits are only deducted after questions are successfully generated — never before. If the AI call fails, the user keeps their credits.
